# Quaternion Utility Library

C++ library for 3D vector & quaternion math utility functions.

## Vec3

- **Vector arithmetic** — addition, subtraction, scalar multiplication, and scalar division.
- **Dot product** — computes scalar product of two vectors.
- **Cross product** — computes cross product to two input vectors.
- **Norm & normalization** — computes vector magnitude and unit vectors.
- **Clamp** — clamps floating-point values to specified range.

## Quaternion

- **Identity quaternion** — represents zero rotation.
- **Quaternion arithmetic** — addition & scalar multiplication.
- **Normalization** — converts quaternion to unit length.
- **Conjugate** — computes quaternion conjugate.
- **Hamilton product** — multiplies two quaternions.
- **Vector rotation** — rotates a 3D vector using a unit quaternion.
- **Quaternion from two vectors** — computes the shortest rotation between two unit vectors.
- **Angle extraction** — computes the rotation angle in radians or degrees.
- **Roll & pitch extraction** — extracts roll and pitch Euler angles from a unit quaternion.
- **Angle conversion** — converts between radians and degrees.
